# TaiwanStockSupplyChainCrawler

## 功能說明 </br>
使用selenium + bs4去爬取PChome股市分類報價清單, 將清單儲存於db</br>
僅爬取下列分頁</br>
上市上櫃-分類報價 https://pchome.megatime.com.tw/group/ </br>
概念類股報價 https://pchome.megatime.com.tw/group/sto3

全部筆數約6000筆, 執行時間1小時
